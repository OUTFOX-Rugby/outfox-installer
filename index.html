OutFox Installer Steps

  1. Close the OutFox game/app if it's open, and have controllers 1 to 7 plugged into the wires. You do not need to turn them on, and keep 8 to 15 powered off.
  2. Extract the outfox-setup.zip file once it's downloaded with right-click → Extract All.
  3. Open the extracted folder and double-click Install-OutFox.bat. Click Yes on the Windows security (UAC) prompt and let it run to the finish screen, then close that window.
  4. Restart the PC.
  5. After it boots, install once more: double-click Install-OutFox.bat again, approve the prompt, let it finish.
  6. Restart the PC again.
  7. Now power on the controllers — wireless ones (8 to 15) first, then the wired ones (1 to 7).
  8. If any controller turns on and then switches off, turn it off and on once more (sometimes one needs a second try) and give it ~30 seconds to settle.
  9. You should reach all 15 connected and stable, with no lights flashing.

If anything's still wrong, send us this one file so we can check:
C:\ProgramData\OutFox\ControllerService\service.log (it appears as just "service" in the Windows folder explorer).
